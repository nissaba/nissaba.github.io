Strategies

A creative tool inspired by Oblique Strategies. Generate thought-provoking prompts to help break through mental blocks in design, music, writing, and beyond.

Key Features

  • Shuffle and reveal curated Oblique Strategies-style prompts
  • Dark mode and distraction-free interface
  • Optional daily notification for a random strategy
  • Local-only app: no internet required
